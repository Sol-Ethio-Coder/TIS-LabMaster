# TIS LabMaster Hub - Teacher Dashboard

A comprehensive attendance and grade management system for teachers with Supabase cloud backup.

## Features

- 📊 **Grade Management** - Track student scores across multiple terms
- ✅ **Attendance Tracking** - Mark present/absent with timestamps
- 📈 **Analytics Dashboard** - Visualize class performance
- 📄 **Report Cards** - Generate individual student reports
- ☁️ **Cloud Backup** - Auto-sync to Supabase
- 🌓 **Dark/Light Theme** - Toggle between themes
- 📥 **Export/Import** - CSV export and JSON backup

## Tech Stack

- HTML5
- CSS3 (with responsive design)
- JavaScript (ES6+)
- Supabase (Cloud storage)
- Font Awesome Icons
